## EFI folder for booting Thinkpad T470 i5-7300u

## Supported OS
 - MacOS Mojave (100% Tested)
 - MacOS Catalina ()

## Working
 - Boot
 - Intel HD620
 - Audio (Local and External)
 - NVMe
 - Brightness control (but too bright for me)
 - Wake from sleep
 - Shutdown 
 - Fan and Temp
